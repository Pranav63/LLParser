Hey,This Project is Console Dependent :)

The intention of the small scaled Project is to learn how to code LL(1) Parser in Javascript and this majorly includes:
1) Finding the first of a Terminal variable of a grammer.
2) Finding the follow of a Terminal variable of a grammer.

Grammer's Production Rules are:
1: S -> F
2: S -> (S + F)
3: F -> a

Similary we will take a more complex Grammer production next :
1: E -> TX
2: X -> +TX
3: X -> ε
4: T -> FY
5: Y -> *FY
6: Y -> ε
7: F -> a
8: F -> (E)
